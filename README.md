# Клон SoundCloud для настольных компьютеров

Настольное приложение, которое имитирует функциональность SoundCloud с помощью Electron и React. Оно использует публичный API SoundCloud для загрузки треков, плейлистов и данных пользователей.
Также работает в странах, где доступ к SoundCloud заблокирован.


## Особенности

- Поиск и воспроизведение треков
- Просмотр плейлистов и профилей пользователей
- Локальное хранилище избранного
- Медиаплеер с элементами управления
- Темы (в том числе пользовательские)
- Язык (ru/eng)
- SoundCloud Sync (лайки, дизлайки треков, плейлистов, просмотр карточек исполнителей, все синхронизируется с вашим профилем).
- Эквалайзер. 

## Настройка

1. Клонируйте репозиторий.
2. Установите зависимости: `npm install`.
3. Запустите приложение: `npm run dev`.
4. Или скачайте релиз.

## Использование API

Использует публичный ID клиента SoundCloud для запросов API. Замените `YOUR_PUBLIC_CLIENT_ID` в коде на ваш фактический ID клиента.

## Технологии

- Electron
- React
- Axios для вызовов API
- Howler.js для воспроизведения аудио
- SQLite для локального хранения






# SoundCloud Desktop Clone

A desktop application that mimics SoundCloud functionality using Electron and React. It uses SoundCloud's public API for fetching tracks, playlists, and user data.
Also works in blocked countries


## Features

- Search and play tracks
- View playlists and user profiles
- Local storage for favorites
- Media player with controls
- Themes (and custom one)
- Language (ru/eng)
- SoundCloud Sync (Like, unlike tracks, playlists, view artist cards, all synchronized with your profile.)
- Equaliser 

## Setup

1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the app: `npm run dev`
4. Or download the release.

## API Usage

Uses SoundCloud public client ID for API requests. Replace `YOUR_PUBLIC_CLIENT_ID` in the code with your actual client ID.

## Technologies

- Electron
- React
- Axios for API calls
- Howler.js for audio playback
- SQLite for local storage
